# 智能产品检索系统使用说明

## 系统概述

智能产品检索系统是基于大语言模型（LLM）和规则引擎的产品匹配工具，帮助销售人员快速根据客户需求匹配产品。

---

## 一、配置 API Key

### 1. 申请大模型 Key

访问宇视内部 AI 平台：**https://tokenhub.uniview.com/**

- 选择 **glm-5** 模型
- 申请完成后获得以下信息：
  - **API Key**（访问密钥）
  - **Base URL**（API 端点地址）
  - **Model**（模型名称，通常为 `glm-5`）

### 2. 填写配置信息

在系统左侧边栏的 **API 配置** 区域填写：

| 配置项 | 说明 |
|--------|------|
| **API Key** | 从 tokenhub 获取的访问密钥 |
| **Base URL** | API 端点地址（例如：`https://aibedrock.uniview.com`） |
| **Model** | 模型名称（填写 `glm-5`） |

填写完成后点击 **保存配置** 按钮，配置将保存在浏览器本地存储中。

## 二、系统操作流程

### 步骤 1：创建产品分类（规则配置页面）

#### 1.1 了解全局配置

进入 **规则配置** 页面 → **全局配置** 标签页：

- **价格排序方向**：控制检索结果的排序方式（`asc` 升序 / `desc` 降序）
- **标签加分规则**：为特定标签的产品加分/减分（如"新品"加 5 分、"清仓"减 3 分）

#### 1.2 创建新产品分类

**推荐方式：通过彩页自动创建分类**

在 **规则配置** 页面顶部找到 **"从文档自动创建分类"** 功能：

1. 上传 **2-4 份** 同类产品的彩页 PDF
2. 点击 **"开始智能分析"**
3. AI 将自动识别：
   - 产品类别名称（如"速通门"、"网络摄像机"等）
   - 类别简介
   - 参数字典结构

#### 1.3 人工审核参数字典

AI 提取完成后会显示参数列表，**建议人工审核要点**：

- ✅ **保留主要参数**：只保留产品间有明显差异的关键规格
- ✅ **参数数量适中**：建议 8-20 个参数，避免过多
- ✅ **参数类型正确**：
  - 数值型（分辨率、温度、重量）→ `numeric`
  - 固定枚举值（接口类型、防护等级）→ `enum`
  - 纯文本描述 → `text`
- ✅ **单位明确**：数值型参数务必填写单位（如 `MP`、`mm`、`℃`）
- ✅ **包含价格参数**：系统会自动添加，确认存在即可

审核完成后点击 **"创建分类"**。

---

### 步骤 2：配置分类参数规则

进入 **规则配置** 页面 → 选择刚创建的分类标签页。

#### 2.1 配置参数字典

在 **参数字典** 区域，可以微调每个参数的属性：

| 配置项 | 说明 |
|--------|------|
| **参数中文名** | 显示名称（如"分辨率"、"防护等级"） |
| **列名** | 数据库列名（英文下划线格式，如 `resolution_mp`） |
| **参数类型** | `numeric` 数值型 / `enum` 枚举型 / `text` 文本型 |
| **单位** | 数值型参数的单位（如 `MP`、`fps`、`℃`） |
| **枚举值** | 枚举型参数的可选值列表（如 `["IP67", "IP66"]`） |
| **默认比较方式** | 参数匹配时的运算符（见下方说明） |
| **提示** | 参数说明（可选） |

#### 2.2 配置比较方式

**默认比较方式** 决定了智能检索时如何匹配参数：

| 比较方式 | 适用场景 | 示例 |
|----------|----------|------|
| `=` | 精确匹配 | 防护等级必须是 IP67 |
| `>=` | 满足最低要求 | 分辨率不低于 5MP，帧率不低于 30fps |
| `<=` | 满足最高限制 | 价格不超过 10000 元 |
| `contains` | 包含匹配 | 供电方式包含 PoE（可以是 DC12V/PoE） |

**配置建议**：
- 数值型规格（分辨率、帧率）→ 建议用 `>=`
- 枚举型规格（防护等级、接口类型）→ 建议用 `=` 或 `contains`
- 价格参数 → 建议用 `<=` 或保持 `=`

#### 2.3 配置特殊规格过滤

在 **特殊规格过滤** 区域填写列名列表（逗号分隔）：

- **作用**：这些参数必须严格匹配，不符合的产品直接过滤掉
- **示例**：`["interface", "shutter_type", "protection_level"]`

#### 2.4 配置重要规格评分

在 **重要规格评分** 区域配置权重：

- **作用**：匹配程度越好，得分越高（0-100 分基础分）
- **格式**：每个参数配置权重（自动归一化）和偏好方向
- **示例**：
  ```json
  {
    "resolution_mp": {"weight": 0.35, "preference": ">="},
    "frame_rate": {"weight": 0.25, "preference": ">="}
  }
  ```

配置完成后点击 **保存配置**。

### 步骤 3：导入产品数据（产品库管理页面）

进入 **产品库管理** 页面。

#### 3.1 选择产品分类

在页面顶部下拉框选择目标分类（如"工业相机"）。

#### 3.2 通过彩页自动提取产品

**推荐方式：上传彩页自动提取**

在 **"从彩页提取产品"** 区域：

1. 上传 **产品彩页 PDF**（一份彩页可包含多个型号）
2. 选择正确的 **产品分类**
3. 点击 **"开始提取"**
4. AI 将自动识别：
   - 产品型号（product_id）
   - 产品名称（product_name）
   - 标签（tags）
   - 所有参数规格

#### 3.3 人工审核提取结果

提取完成后显示预览表格，**人工审核要点**：

- ✅ **型号唯一**：每个型号必须是独立的行
- ✅ **数值正确**：确认数值型参数已去除单位（如 `5.0` 而不是 `5MP`）
- ✅ **枚举匹配**：枚举值应与参数字典中的选项一致
- ✅ **null 处理**：文档未提及的参数填 `null`（不是空字符串）
- ✅ **标签合理**：标签用分号分隔（如 `新品;现货`）

#### 3.4 写入产品库

审核完成后选择写入方式：

- **追加写入**：将新产品追加到现有产品库（重复型号会被更新）
- **覆盖写入**：替换该分类下的全部产品数据

点击相应按钮完成导入。

#### 3.5 其他导入方式

**CSV/Excel 导入**：

- 准备 CSV 文件（UTF-8 编码，第一行为列名）
- 必需列：`product_id`、`product_name`、`tags`、`price`
- 规格列：与参数字典中的 `column` 名称一致
- 点击 **"上传 CSV 或 Excel 文件"** 导入

**下载当前产品表**：

- 点击 **"下载【分类】产品表 CSV"** 导出当前产品数据
- 可用于备份或批量编辑后重新导入

---

### 步骤 4：智能检索

进入 **智能检索** 页面。

#### 4.1 输入客户需求

在文本框中粘贴客户提供的竞品参数或需求描述，例如：

> 需要一款彩色工业相机，分辨率不低于500万像素，帧率至少25fps，
> 接口要求GigE，需要全局快门，工作温度-20到70度，防护等级IP67

也可点击右侧示例按钮快速填入示例文本。

#### 4.2 开始检索

点击 **"🚀 开始智能检索"** 按钮，系统将自动执行：

**Step 1a - 产品类别识别**
- Claude 分析客户需求文本
- 自动识别最匹配的产品分类
- 显示识别结果和置信度

**Step 1b - 参数提取**
- Claude 根据分类参数字典提取结构化参数
- 显示提取的参数列表和需求摘要

**Step 2 - 规则引擎筛选与评分**
- 加载该分类的产品库
- 应用特殊规格过滤（不符合的产品被排除）
- 应用重要规格评分（匹配度越高得分越高）
- 应用标签加分规则
- 按价格排序

#### 4.3 查看检索结果

检索完成后显示：

- **推荐产品列表**：按得分排序，显示总分和各项得分明细
- **过滤原因**：显示被过滤产品的原因（参数不匹配）
- **需求摘要**：客户需求的简化总结
- **参数对比**：产品参数与需求参数的对比视图

## 三、常见问题与优化

### Q1：检索结果不准确怎么办？

**检查参数比较方式**：

进入 **规则配置** 页面 → 选择分类 → 调整 **默认比较方式**：

- 如果客户要求"不低于"，应设置为 `>=`
- 如果客户要求"包含"，应设置为 `contains`
- 如果客户要求"精确匹配"，应设置为 `=`

### Q2：某些产品被错误过滤？

**检查特殊规格过滤**：

- 特殊规格要求严格匹配，稍有差异就会被过滤
- 如果客户需求不够明确，可以减少特殊规格字段数量

### Q3：如何调整产品优先级？

**方法 1：调整重要规格权重**

在规则配置页面调整参数权重，权重越高影响越大。

**方法 2：调整标签加分规则**

在全局配置中为特定标签加分（如"新品"加 5 分、"爆款"加 4 分）。

### Q4：彩页提取不准确？

**人工审核建议**：

- 确认 PDF 文件清晰度（建议 600DPI 以上）
- 确认彩页包含完整的规格参数表
- 提取后务必人工审核数值和枚举值
- 可以多次提取，合并结果

### Q5：如何批量修改产品数据？

1. 进入产品库管理页面
2. 下载当前产品表 CSV
3. 用 Excel 批量编辑
4. 重新上传 CSV 导入（选择覆盖写入）

## 五、最佳实践建议

### ✅ 产品分类管理

- 每个分类保持 10-50 款产品（不宜过多）
- 参数字典包含主要差异参数即可（8-20 个）
- 定期更新产品库，删除停产型号

### ✅ 参数字典设计

- 数值型参数优先用 `>=` 比较方式
- 枚举型参数明确所有可选值
- 单位保持一致（如分辨率统一用 MP）
- 参数命名语义化（如 `resolution_mp`）

### ✅ 彩页提取流程

- 选择清晰完整的彩页（包含规格参数表）
- 一份彩页可包含多个型号
- 提取后必须人工审核
- 定期更新产品数据

### ✅ 检索结果优化

- 根据实际效果调整参数比较方式
- 合理设置特殊规格过滤（不宜过多）
- 调整重要规格权重突出关键参数
- 使用标签加分突出重点产品
